# 🧪 Solar-Coders Manual Testing Project

## 📌 Project Overview
Solar-Coders Manual Testing Project is a comprehensive manual testing project conducted on an eCommerce web application to validate its functionality, usability, reliability, and overall user experience.  
The project covers multiple modules including Login, Registration, Product Search, Add to Cart, Wishlist, Checkout, and Home Page testing with detailed test scenarios, test cases, bug reports, and summary documentation.

This project demonstrates practical knowledge of Software Testing Life Cycle (STLC), defect reporting, test planning, and real-world QA testing practices.

---

# 🎯 Objectives

- Validate all major functionalities of the eCommerce application
- Identify defects and usability issues
- Ensure smooth user experience across modules
- Prepare professional QA documentation
- Improve software quality through structured testing
